
Compreensão de texto != Interpretação de texto

Compreensão de texto é a decodificação da mensagem

`Compreensão = decodificação = análise do que está explícito no texto.

Já a interpretação é uma passo a mais.

``Interpretação = conclusão = inferência = dedução``

Compreensão é para **Dentro** do texto
Interpretação é para **Fora** do texto

Para compreender eu me prendo ao texto pra interpretar eu me extrapolo o texto

`Compreender = Preso ao texto / Interpretar = Extrapolo o texto`

### Tópicos importantes para auxiliar a compreensão do texto.

1. Quem é o autor do texto?
2. O que sugere o título do texto?
3. Qual a intenção daquele texto?
4. De onde foi tirado o texto?
5. A Linguagem utilizada no texto

### Diferenciar o Texto do Verbal x não verbal x misto

***Texto verbal*** - utiliza-se palavras
***Texto não verbal*** - não se utiliza de palavras
***Texto misto*** - utiliza-se de palavras, imagens cores e outras expressões de linguagem

Quando aparecer Inferir, deduzir, concluir estamos referindo a uma interpretação de texto

***Inferência*** - é o processo pelo qual, com base em determinados dados, chega-se a uma conclusão.

***Inferir:*** Concluir, deduzir

***Deduzir:*** Concluir, inferir

***Concluir:*** Chegar a um resultado, decisão, afirmação
