[[Gramática I]]
## Object Pronouns

Exercem a Função de **sujeito** da oração

| Singular                            | Exemplos:                                                  |
| ----------------------------------- | ---------------------------------------------------------- |
| **Me** -  me, mim                   | Do you like **me**? / Você gosta de mim?                   |
| **You** - te, ti ,o, a, lhe, a você | I love **you** / Eu te amo.                                |
| **Him** - o,lhe, a ele              | I love **him** very much. / Eu o amo muito                 |
| **Her** - a,lhe, a ela              | Saw **her** yesterday. / Eu a vi ontem                     |
| **It** - o,a,lhe                    | You loves **it**. / Você lhe adora.                        |
| **Plural**                          |                                                            |
| **Us** - nos                        | They never talk to **us**. / Eles nunca conversam conosco. |
| You - vos, os, as, lhes, a vocês    | She writes to **you**. / Ela escreve para vocês            |
| Them - os, as , lhes                | She talked to **them**. / Ela Falou com eles.              |
