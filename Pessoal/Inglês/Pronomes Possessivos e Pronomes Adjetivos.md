
| Significado                                | Adjetivo Possessivos | Pronomes Possessivos |
| ------------------------------------------ | :------------------: | :------------------: |
| meu(s), minhas(s)                          |          My          |         Mine         |
| teu(s),tua(s),seu(s),sua(s),de você(s)     |         Your         |        Yours         |
| dele(s), seu(s), sua(s)                    |         His          |         His          |
| dela(s),seu(s), sua(s)                     |         Her          |         Hers         |
| dele(s), dela(s),seu(s),sua(s)             |         Its          |         Its          |
| nosso(s), nossa(s)                         |         Our          |         Ours         |
| vosso(s),vossa(s),seu(s), sua(s), de vocês |         Your         |        Yours         |
| dele(s),dela(s),seu(s),sua()               |        Their         |        Theirs        |
Exemplos:

|                  Pronomes Possessivos                   |            Adjetivos Possessivos             |
| :-----------------------------------------------------: | :------------------------------------------: |
|       This house is **mine**<br>Esta casa é minha       |  This is **my** house.<br>Esta é minha casa  |
|      This house is **yours**.<br>Esta casa é sua.       | This is **your** house<br>Esta é a sua casa  |
|        This house is** his**<br>Esta casa é dele        | This is **his** house.<br>Esta é a sua casa  |
|       This house is **hers**<br>Esta casa é dela        |  This is **her** house<br>Esta é a sua casa  |
|        This place is **its**<br>Este lugar é seu        | This is **its** place<br>Este é o seu lugar  |
|  These houses are **ours**<br>Estas casas são nossas.   | This is **our** house<br>Esta é a nossa casa |
| These houses are **yours**<br>Estas casas são de vocês. | This is **your** house<br>Esta é a sua casa  |
|  These houses are **theirs**<br>Estas casas são deles   | This is **their** house<br>Esta é a sua casa |
## Atenção

**Em inglês, os pronomes possessivos têm a mesma forma no singular e no plural.**

para que você fixe melhor seu aprendizado, indicamos que faça anotações de todos os pronomes que está aprendendo no curso, para treiná-los posteriormente.

Quanto mais exercitar o que aprendeu, mais rápido aprenderá o novo idioma
