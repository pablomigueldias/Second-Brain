[[Gramática I]]
## Subject Pronouns

Exercem a Função de **sujeito** da oração

| Singular                                  | Exemplos:                                                            |
| ----------------------------------------- | -------------------------------------------------------------------- |
| **I** - eu                                | **I** am a student / Eu sou estudante.                               |
| **You** - tu, você                        | **You** are beautiful / Você é bonita.                               |
| **He** - ele                              | **He** lives in São Paulo. / Ele mora em São Paulo                   |
| **She** - ela                             | **She** is Brazilian / Ela é Brasileira.                             |
| **It** - ele, ela (para coisas e animais) | **It** is a table. / É uma mesa.                                     |
| **Plural**                                |                                                                      |
| **We** - Nós                              | **We** speak Portuguese in Brazil. / Nós falamos português no Brasil |
| **You** - Vós, vocês                      | **You** can go together. / Vocês podem ir juntos.                    |
| **They** - eles, elas                     | They are intelligent. / Eles são inteligentes                        |
