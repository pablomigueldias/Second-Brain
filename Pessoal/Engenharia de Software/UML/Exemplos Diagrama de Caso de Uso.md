Exemplo de Diagrama de Casos de Uso: Sistema Bancário
Sistema permite a abertura e encerramento de contas.

Também permite consultas e saldo e movimentação, além de realizar saques, depósitos e transferências.

Contas do tipo Corrente ou Poupança

Interagem com o sistema: Cliente Pessoa Física ou Jurídica Funcionários como caixas e gerentes do banco.

1- identificar os Atores

- Cliente
	- Cliente pode ser pessoa física ou pessoa jurídica
- Funcionário
	- O funcionário pode ser Caixa ou Gerente

2- Identificar Casos de Uso

- Tirar Extrato
- Abrir Conta Corrente = ok
- Abrir Conta Poupança = ok
- Realizar Saque = ok
- Fazer Depósito = ok
- Encerrar Conta = ok
- Tirar Saldo = ok
- Registrar Transação = ok
