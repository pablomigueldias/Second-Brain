**Definição**: uma função $f:\mathbb{R} \rightarrow \mathbb{R}$  chama-se quadrática quando existem números reais *a*, *b* ,*c*, com $a \neq  0$, tal que $f(x) = ax^{2} + bx + c$  para todo $x$ $\epsilon$ $\mathbb{R}$


Exemplos:

a.   $f(x) = 3x^{2}-4x+2$

`a = 3 | b = -4 | c = 2`

b.  $f(x)=x^{2}+3x$

`a = 1 | b = 3 | c = 0`

c.  $y=-x^{2}-5$

`a = -1 | b = 0 | c = -5`

d.  $y=-2x^{2}$

`a = -2 | b = 0 | c = 0`


## Gráfico

O gráfico de uma função quadrática, $f(x)=ax^{2}+bx+c$, com $a\neq0$, é uma curva chamada **parábola**.

Exemplos:

a. $f(x) = x^{2}-1$

| $x$ | $f(x)$ |
| :-: | :----: |
| -2  |   3    |
| -1  |   0    |
|  0  |   -1   |
|  1  |   0    |
|  2  |   3    |
$$
\begin{aligned}
f(-2)=(-2)^{2}-1=3\\
f(-2)=(-1)^{2}-1=0\\
f(-2)=(0)^{2}-1=-1\\
f(-2)=(1)^{2}-1=0\\
f(-2)=(2)^{2}-1=3\\
\end{aligned}
$$
